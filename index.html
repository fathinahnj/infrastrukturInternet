Bagaimana cara Internet bekerja?

Internet adalah jaringan global yang menghubungkan jutaan perangkat di seluruh dunia.

Internet bekerja dengan menghubungkan perangkat di seluruh dunia melalui jaringan yang saling terhubung. Data dikirim dalam bentuk paket-paket kecil menggunakan protokol TCP/IP. Setiap perangkat memiliki alamat IP unik untuk mengidentifikasi tujuan pengiriman data. Saat pengguna mengakses suatu layanan online, perangkat mengirimkan permintaan ke server, yang kemudian mengirimkan data yang diminta kembali ke perangkat. Domain Name System (DNS) berperan dalam menerjemahkan nama domain menjadi alamat IP sehingga perangkat bisa menemukan server yang tepat.

HTTP dan HTTPS

Apa itu HTTP?

Hypertext transfer protocol (HTTP) adalah sebuah protokol atau seperangkat aturan komunikasi untuk komunikasi klien-server. Saat Anda mengunjungi situs web, peramban Anda mengirimkan permintaan HTTP ke server web, yang merespons dengan HTTP.

Sumber : https://aws.amazon.com

Apa itu HTTPS?

Hypertext Transfer Protocol Secure (HTTPS) adalah versi atau perpanjangan HTTP yang lebih aman. Di HTTPS, peramban dan server akan membuat koneksi yang aman dan terenkripsi sebelum mentransfer data.

Sumber : https://aws.amazon.com

Bagaimana cara kerja HTTP?

Permintaan Klien: Browser mengirimkan permintaan HTTP ke server (misalnya, untuk mengakses halaman web).
Pengiriman Data: Permintaan dan data dikirim dalam format teks biasa tanpa enkripsi.
Respons Server: Server merespons dengan mengirimkan data yang diminta (seperti halaman HTML) kembali ke browser.
Transfer Data: Data dipindahkan dari server ke klien melalui jaringan, namun tidak ada jaminan keamanan.

Bagaimana cara kerja HTTPS?

Koneksi Aman: Browser mengirimkan permintaan ke server HTTPS untuk membangun koneksi yang aman.
Proses SSL/TLS Handshake: Browser dan server bertukar sertifikat enkripsi (SSL/TLS) untuk mengenkripsi komunikasi.
Enkripsi Data: Permintaan dan respons dienkripsi sehingga hanya pengirim dan penerima yang dapat membacanya.
Transfer Aman: Data dipindahkan melalui jaringan dengan enkripsi, melindungi dari penyadapan dan manipulasi.

Bagaimana cara nama domain bekerja?

Nama domain adalah alamat unik yang digunakan untuk mengidentifikasi situs web di internet.

Nama domain bekerja sebagai alamat yang mudah diingat untuk mengakses situs web di internet, menggantikan alamat IP yang kompleks. Ketika pengguna mengetik nama domain di browser, permintaan tersebut dikirim ke server DNS (Domain Name System), yang bertugas menerjemahkan nama domain menjadi alamat IP yang sesuai. Server DNS kemudian mencari alamat IP yang terkait dengan nama domain tersebut, dan mengembalikannya ke browser. Setelah mendapatkan alamat IP, browser dapat terhubung ke server yang menyimpan situs web tersebut, memungkinkan pengguna untuk mengakses konten yang diinginkan dengan mudah. Proses ini terjadi dalam hitungan detik, sehingga pengguna tidak perlu mengingat atau memasukkan alamat IP secara manual.

Sumber: Techtarget

Hosting bekerja dengan menyediakan ruang di server untuk menyimpan file dan data sebuah situs web, sehingga situs tersebut dapat diakses secara online. Ketika pengguna mengetik nama domain di browser, permintaan diarahkan ke server hosting tempat file situs web disimpan. Server kemudian mengirimkan data situs tersebut ke browser, yang menampilkannya kepada pengguna. Hosting memungkinkan situs web tetap tersedia 24/7 di internet.

Sumber: Squarespace Help Center

Bagaimana cara hosting bekerja?

Hosting adalah layanan yang menyediakan tempat di server sehingga situs tersebut dapat diakses secara online.

Bagaimana cara DNS bekerja?

Domain Name System adalah sistem yang berfungsi seperti buku alamat internet.

Sumber: Amazon Web Services

Pengguna mengetik URL di browser
Permintaan dikirim ke DNS resolver
Resolver memeriksa DNS yang sesuai di cache lokal
Jika tidak ditemukan, permintaan diteruskan ke server DNS root
Server DNS root mengarahkan permintaan ke server DNS TLD
TLD server mengarahkan permintaan ke server DNS otoritatif untuk domain tersebut
Server otoritatif mengembalikan alamat IP dari domain yang diminta
Browser menggunakan alamat IP untuk menghubungi server web dan menampilkan situs

Bagaimana cara browser bekerja?

Browser adalah perangkat lunak yang digunakan untuk mengakses, menampilkan, dan berinteraksi dengan konten di internet melalui protokol web seperti HTTP atau HTTPS.

Sumber: BrowserStack

Browser bekerja dengan mengirimkan permintaan HTTP atau HTTPS ke server setelah pengguna memasukkan URL atau mengklik tautan. Permintaan ini dikirim ke server yang menyimpan situs web yang diminta. Server kemudian merespons dengan mengirimkan file berupa HTML, CSS, JavaScript, dan konten lain yang diperlukan. Browser menerima file tersebut, memprosesnya, dan menampilkan halaman web kepada pengguna. HTML menentukan struktur konten, CSS mengatur tampilannya, dan JavaScript mengelola interaksi dinamis di halaman. Selain itu, browser juga mengelola caching, cookies, dan keamanan untuk memastikan pengalaman berselancar yang lancar dan aman.
